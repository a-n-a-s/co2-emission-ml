About The Page

This website is used to predict the CO2 emission by a vehicle with the details of Engine Size , Cylinders and Fuel.
This page uses a ML model and flask as backend.
The ML model is trained on 1607 types of different vehicle types and model dataset.
You can check the dataset here.
Python is used for ML . Packages like scikit-learn , pandas , numpy are used .
The Training of the model is done through LinearRegression .
Pickle module is used to convert the ml.py to model.pkl so that it can be used in website.
Flask is used as backend and the website is hosted on Heroku.
Html5 and Tailwind CSS is for frontend.
